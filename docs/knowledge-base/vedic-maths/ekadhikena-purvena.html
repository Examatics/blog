एकाधिकेन पूर्वेण

Ekādhikena Pūrvena - “By one more than the previous one”

A Vedic math shortcut for squaring numbers ending in 5

Step-by-Step: Squaring 35

  • Split the number → 3
  • Add one → 3 + 1 = 4
  • Multiply → 3 x 4 = 12
  • Append 25 → Final answer = 1225

How It Works

Any number ending in 5 can be expressed as (10x + 5). Squaring it gives:

(10x + 5)² = 100x(x + 1) + 25

This shortcut is a clever rearrangement of algebra, made intuitive through pattern recognition.

Try It Yourself

  • 45² → 4 x 5 = 20 → 2025
  • 85² → 8 x 9 = 72 → 7225
  • 105² → 10 x 11 = 110 → 11025

Why Students Love It

  • No need to memorize large multiplication tables
  • Builds confidence in mental math
  • Encourages pattern-based thinking
  • Perfect for competitive exams and quick calculations
